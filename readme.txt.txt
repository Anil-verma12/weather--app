# Weather App

## Description
This project is a simple weather application that allows users to:
- View the **current temperature**.
- See the **minimum and maximum temperature** for the day.
- Display the **city name**.
- Search for the temperature by entering a city name.

## Features
- Fetches real-time weather data.
- User-friendly interface.
- Displays essential weather details.
- Search functionality to get weather details for any city.

## Technologies Used
- HTML, CSS, Bootstrap (Frontend)
- Python, Flask (Backend)
- OpenWeather API 

## Installation & Setup
1. Clone the repository:
   ```sh
   git clone https://github.com/yourusername/weather-app.git
   ```
2. Navigate to the project folder:
   ```sh
   cd weather-app
   ```
3. Install required dependencies:
   ```sh
   pip install -r requirements.txt
   ```
4. Run the application:
   ```sh
   python app.py  # or python manage.py runserver (for Django)
   ```
5. Open your browser and go to `http://localhost:5000/` (Flask) or `http://127.0.0.1:8000/` (Django).

## Usage
1. Open the application.
2. Enter the city name in the search bar.
3. Click the search button to fetch weather details.
4. View the current temperature, minimum and maximum temperature, and city name.
